    李修道看出了王大猛眼中的关爱，也就直接坐在石桌旁，拿起来个卤制的猪肘啃了起来，猛然觉得这肘子的味道胜过他吃过的一切美食。

    “喝点？”

    王大猛笑着指了指旁边有些浑浊的米酒。

    “不了，不了，等下还要打坐修炼。”

    李修道立刻摇头拒绝，他不知道这具身体酒量的程度，若是喝多了说了些胡话反倒不妙。

    “你说的也有道理，长生之路片刻不能怠惰。”

    王大猛眼中带着追忆感叹一声。

    “王大哥，此言有理，有道是修行如逆水行舟，不进则退。”

    “哈哈哈哈...”

    王大猛听到这句话忽然大笑起来，直到十几秒后才渐渐的止住笑声。

    “看你今天修炼，对武道也有兴趣？”
